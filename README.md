<h1>multiple-app</h1>
<p>複数のアプリを扱うことができます。実装予定であり、カレンダー(ToDo)、アラーム、電卓、フォルダ管理、メモなどといった複数の機能が使えるようになります。</p>

<h2>今後の課題</h2>
<ul>
  <li>ToDoの内容を保持</li>
  <li>アラーム機能実装</li>
  <li>電卓サイズ調整</li>
  <li>フォルダ管理実装</li>
  <li>複数アプリをサイドバーで+を押すと機能追加、-を押すと機能削除を実装</li>
  <li>メモ機能実装</li>
</ul>
